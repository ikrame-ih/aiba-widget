# Aiba Product

Desktop productivity companion with a geisha-inspired character named **Aiba**.

## Appearance

- **Dark mode default** — Tricorn Black base, Marooned cards, Verbena text, Keystone metadata, Ceremonial Gold accent.
- **Light mode** — toggle in Preferences (`settings.theme`).
- **Single accent rule** — gold for progress, actions, and flow; Positive Red only for guard/critical states.

## Phases by time (3 modes preserved)

| Phase             | Hours       | Primary action       | Ambient shift                |
| ----------------- | ----------- | -------------------- | ---------------------------- |
| Morning · Plan    | 6:00–17:59  | Outcome + priorities | Cooler surface, Verbena tint |
| Afternoon · Focus | 18:00–20:59 | Protected block      | Deepest black, gold accent   |
| Evening · Close   | 21:00–5:59  | Wrap up              | Marooned warmth, muted       |

Focus is reachable from any phase. Preview any phase manually; return to auto anytime.

## Palette (Sherwin-Williams — depurada)

| Color           | Hex       | Role           | Contrast vs Tricorn |
| --------------- | --------- | -------------- | ------------------- |
| Tricorn Black   | `#2F2F30` | Base           | —                   |
| Marooned        | `#4E3132` | Cards/panels   | Subtle separation   |
| Verbena         | `#F3E0E0` | Primary text   | ~10.4:1 (AAA)       |
| Keystone Gray   | `#9E9284` | Secondary text | ~4.4:1 (AA large)   |
| Ceremonial Gold | `#D69E59` | Accent / Ensō  | ~5.7:1              |
| Positive Red    | `#AD2C34` | Guard only     | Critical            |

**Removed from UI:** Coral Bead, Grenadine, Raucous Orange, Sierra Redwood (visual noise / poor contrast).

## Widget sizes

- **Compact** 400×480 — timer, status, expand / hide
- **Expanded** 900×720 — Aiba sidebar + Plan, Focus, Unwind, Preferences

## Focus progress — Ensō

During an active block or recovery, a circular sumi-e ring (Ensō) shows progress. The timer sits centered inside the ring; stroke uses Ceremonial Gold. Respects reduced motion.

## Deep work features

1. Session machine: prepare → focus → pause → recovery → review
2. Fine-tune focus (skill/challenge) in progressive disclosure
3. Quick notes: `openLoops` + `brainDump`
4. Sessions: list always; pattern cards after 3 blocks
5. Preferences: theme, language, reduced motion, tunnel vision, guard
6. Ask Aiba: local ES/EN help

## Privacy

All data and help resolution stay on device.
